Machine Learning Content

  •    How to begin an ML project
  •    K-nearest-neighbors classifier
  •    Support Vector Machines + Linear/Logistic Regression
  •    Decision Trees
  •    Naive Bayes

Prerequisites:

  • Basic Python knowledge (CSE 20)
  • Proficiency in basic Algebra (we will explain any Linear Algebra / Calculus)
  • Access to any computer

Frameworks We'll Use:

  • NumPy adds support for large matrices along with mathematical functions on them
  • SciKit Learn is a machine learning library with implementations of many algorithms and a rich ecosystem


Deep Learning Content

  •    Intro to Neural Networks and PyTorch
  •    Computer Vision: Convolutional Neural Networks
  •    Natural Language Processing: Recurrent Networks + Attention
  •    Reinforcement Learning

Prerequisites:

  • NumPy + Python proficiency
  • Basic Calculus, Linear Algebra, Algebra proficiency
  • Access to any computer

Frameworks We'll Use:

  • PyTorch is a library useful for prototyping and developing deep learning models
  • OpenCV contains a set of tools to manipulate image files, often used to preprocess + augment CV data
  • PIL is the standard library to save/load images into formats understandable by PyTorch and other Python tools
  • Spacy is filled with pipelines/tools to manipulate text and sequence data, extremely useful in NLP preprocessing
  • HuggingFace Transformers is an open source model zoo with many pretrained language models
  • OpenAI Gym is a framework released by OpenAI to make running reinforcement learning experiments easier
